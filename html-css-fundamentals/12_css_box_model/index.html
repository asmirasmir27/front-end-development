U prethodnim lekcijama ilustrovani su osnovni postulati CSS jezika. Definisani su pojmovi interne, eksterne i linijske stilizacije, kao i pojmovi CSS selektora i opisa. Demonstrirano je i korišćenje osnovnih CSS selektora. Predmet lekcije pred vama jeste još jedan u nizu najznačajnijih CSS pojmova. Reč je o CSS Box modelu.
